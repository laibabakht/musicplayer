# JavaScript Music Player

A responsive and interactive music player built with **HTML, CSS, and JavaScript**. The application allows users to play, pause, skip, and navigate through songs with an intuitive user interface. It features a dynamic progress bar, volume control, playlist support, and real-time track information, providing a smooth and enjoyable music listening experience.

## Features

* Play, pause, next, and previous track controls
* Interactive progress bar with seek functionality
* Volume control and mute option
* Responsive design for desktop and mobile devices
* Dynamic song title, artist, and album artwork display
* Playlist support
* Clean and modern user interface

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

This project demonstrates core JavaScript concepts such as DOM manipulation, event handling, audio APIs, and responsive UI development.
